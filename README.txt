【写真の差し替え方法】★コード編集不要★
────────────────────────────────────────
images フォルダの写真を「同じファイル名」で上書きするだけ。

  hero.jpg          1500×1900   TOP 全画面
  intro.jpg         1000×1400   （予備）
  wide-street.jpg   1600×900    TOP/Access 横長バンド
  hero-stay.jpg     1500×1900   Stay ヘッダー
  stay-1〜5.jpg     縦1000×1400 / 900×1200  Stay 各所
  stay-wide.jpg     1600×900    Stay 横長
  hero-books.jpg    1500×1900   Books ヘッダー
  books-1〜5.jpg    縦1000×1400 / 900×1200  Books 各所
  books-wide.jpg    1600×900    Books 横長
  hero-access.jpg   1600×1100   Access ヘッダー
  og.jpg            1200×630    SNSシェア用

  ※多少サイズがズレても中央表示で自動調整されます
  ※比率（縦長/横長）だけ合わせればOK
  ※1枚300KB以下が理想。squoosh.app で圧縮推奨

────────────────────────────────────────
【公開方法】GitHub Pages
────────────────────────────────────────
  既存リポジトリ sassa-osaka.github.io の
  全ファイルを、この新しい中身で置き換えます。
  （古いindex.html等を削除 → 新しい中身をアップロード →
   imagesフォルダもアップロード → Commit changes）
  数分で https://sassa-osaka.github.io/ に反映されます。
